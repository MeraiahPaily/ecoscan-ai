🌱 EcoScan AI

EcoScan AI is an AI-powered web application designed to simplify waste classification and promote sustainable living. By leveraging image recognition, the platform enables users to identify different types of waste and make informed disposal decisions, contributing to more effective recycling practices.

🚀 Key Features
AI-Based Waste Detection
Users can upload images of waste items, which are analyzed using intelligent models to determine their category.
Automated Waste Classification
Accurately categorizes materials into recyclable, organic, and non-recyclable waste streams.
User-Centric Interface
Built with a clean, responsive design to ensure accessibility and ease of use across devices.
Real-Time Processing
Provides quick and efficient results, enhancing usability in practical, everyday scenarios.
Scalable Backend Integration
Seamlessly integrates with backend services to manage data efficiently and support future scalability.
🛠️ Tech Stack
Frontend: React with TypeScript
Build Tool: Vite
Styling: Tailwind CSS with shadcn/ui
Animations: Framer Motion
State Management & Data Fetching: React Query
Backend & Database: Supabase
Testing: Vitest and Playwright
📂 Project Overview

The application is structured to ensure modularity and maintainability. It follows a component-based architecture with dedicated layers for UI components, application pages, reusable hooks, and backend integrations. This structure supports scalability and clean code practices, making it suitable for production-level enhancements.

⚙️ Setup & Usage

To run the project locally, clone the repository, install the required dependencies, configure environment variables for backend services, and start the development server. The application will then be accessible in a local development environment.
